formatos de imagen para web

Los formatos

Para la web existen 4 formatos de imágenes: jpg, png, gif y svg. Cada formato de imagen tiene sus ventajas y desventajas propias. Cuando subimos una imagen a nuestra web, le vamos a dar más importancia que se suba lo más rápido posible, y dejando de lado que tenga una excelente calidad, ya que el destino final será la web y no la impresión de dicha imagen. Optimizar una imagen significa guardarla en el formato más conveniente y logrando el menor peso posible del archivo final. El peso de los archivos en la web es importantísimo ya que influye en la velocidad de descarga del archivo en la web.

Formato JPG:


El formato JPG es el indicado para guardar imágenes fotográficas en la web ya que admite millones de colores (24 bits) y mantiene el modo de color en RGB de la imagen original. Obviamente es un formato de archivo que comprime mucho la calidad de la imagen, entonces conviene comprimir lo más posible mientras la calidad de la imagen sea aceptable.

Formato PNG:


La extensión PNG es la más utilizada en marcas para diseñar su logotipo e iconos. La gran ventaja de este formato es que permite guardar imagenes con transparencia, es decir con el fondo transparente.

Formato GIF:


El GIF comprime mejor las imágenes con solidos de color originados en dibujos vectoriales, y tiene una compresión aún mayor que el JPG, y al igual que este la paleta se reduce a 256 colores teniendo una cantidad de colores menor que el JPG. Las imagenes GIF además pueden ser transparentes y animadas.

Formato SVG:


Es el único formato vectorial y por lo tanto escalable para web.Es escalable, pesa poco y permite una definición mayor a tamaños reducidos, mucho mayor que los archivos bitmap. Las ilustraciones vectoriales que posteriormente son animadas con otro programa se suelen guardar en este formato.
